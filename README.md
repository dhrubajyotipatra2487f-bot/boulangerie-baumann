# Boulangerie Baumann — Landing Page

A fast, mobile-first single-page website for **Boulangerie Baumann** (Lausanne) — artisan du Terroir Vaudois depuis 1966.

- Single self-contained `index.html` — no build step, no dependencies, all images inlined.
- Proud Vaudois art direction (green / cream / gold), modernized rocket-baker mascot, terroir story, two Lausanne shops.

## Deploy

> This repo is currently **private**. Free GitHub Pages needs a **public** repo — either make it public (Settings -> General -> Change visibility) or use Cloudflare Pages, which works with private repos.

**Option A - Cloudflare Pages (recommended: free, commercial-friendly, works with private repos):**
1. Cloudflare dashboard -> Workers & Pages -> Create -> Pages -> Connect to Git -> this repo.
2. Framework preset: **None**. Build command: *(leave empty)*. Output directory: **/**.
3. Deploy, then add the custom .ch domain when ready.

**Option B - GitHub Pages (make the repo public first):** Settings -> Pages -> Source: `main` / root.

**Option C - Open locally:** just open `index.html` in any browser.
